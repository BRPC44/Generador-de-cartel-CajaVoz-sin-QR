# Generador de Cartel de Cobro — CajaVoz

Cartel imprimible en formato **A5 horizontal** para comercios que usan **CajaVoz**, la app Android que lee los cobros digitales en voz alta.

## 📋 Contenido

| Archivo | Descripción |
|---|---|
| `index.html` | Cartel listo para imprimir (A5 horizontal) |
| `cajvoz_logo.png` | Logo de CajaVoz |

## 🖨️ Cómo usarlo

1. Abrí `index.html` en **Google Chrome**
2. Completá a mano los campos del cartel (alias de cobro y titular de cuenta)
3. Presioná el botón **"🖨️ Imprimir A5"** o `Ctrl+P`
4. Configurá: tamaño **A5**, orientación **horizontal**, márgenes **ninguno**
5. Guardá como PDF o imprimí directamente

## ✏️ Personalización

Editá `index.html` con cualquier editor de texto para cambiar:
- Número de WhatsApp de contacto
- Texto del footer
- Colores del cartel

## 📱 Sobre CajaVoz

CajaVoz es una app Android que anuncia en voz alta cada cobro recibido por Mercado Pago.  
Ideal para comercios con ruido ambiente donde no se escucha la notificación del teléfono.
